# プレミアムマーケット - 会員限定ECサイト

有料会員限定のオンラインマーケットプレイスです。厳選された商品と信頼できる出品者が集まる、プレミアムなショッピング体験を提供します。

## 🎯 プロジェクト概要

このプロジェクトは、以下の要件を満たす完成度90%以上のマーケティングプラットフォームです：

### 主要機能

#### ✅ 会員管理システム
- 月額料金の自動/手動支払い対応
- 3つの料金プラン（スタンダード・プレミアム・ビジネス）
- 有効会員のみログイン可能
- 会員ステータス管理

#### ✅ 決済システム
- クレジットカード決済対応
- セキュアな決済処理
- 自動課金システム

#### ✅ 商品管理
- 管理者・出品者専用ページ
- 商品登録・編集・削除機能
- 在庫管理
- カテゴリー分類

#### ✅ 通知システム
- 購入時の自動メール通知（購入者・出品者）
- 注文ステータス更新通知
- 会員登録ウェルカムメール

#### ✅ レスポンシブデザイン
- スマートフォン完全対応
- タブレット対応
- デスクトップ対応
- 文字ずれなし、完璧なレイアウト

## 🚀 セットアップ方法

### 1. SCSSのコンパイル

```bash
# Sassをインストール（未インストールの場合）
npm install -g sass

# SCSSをCSSにコンパイル
sass css/style.scss css/style.css --no-source-map

# 監視モード（開発時）
sass --watch css/style.scss:css/style.css
```

### 2. ローカルサーバーの起動

```bash
# Python 3の場合
python -m http.server 8000

# Node.jsの場合
npx http-server -p 8000
```

ブラウザで `http://localhost:8000` にアクセスしてください。
